# Andares Tours World

Sitio web oficial de **Andares Tours World**, agencia de viajes y turismo especializada en:

- Tickets aéreos
- Tours nacionales
- Traslados turísticos
- Hoteles
- Alquiler de movilidades

---

## ✨ Tecnologías

- Next.js
- React
- CSS Utility System
- Design Tokens
- Responsive Design
- SEO Optimization

---

## 🚀 Instalación

Clona el proyecto:

```bash
git clone https://github.com/tuusuario/andares-tours.git
```

Ingresa al proyecto:

```bash
cd andares-tours
```

Instala dependencias:

```bash
npm install
```

Inicia el servidor de desarrollo:

```bash
npm run dev
```
